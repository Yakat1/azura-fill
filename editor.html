Cómo usar esta herramienta y palabras clave
▾

Azura AutoFill Studio agiliza el llenado de somatometría y notas clínicas en Azura.

💡 Palabras clave para Parseo

El sistema reconoce los siguientes términos (insensible a mayúsculas):

🌡️ Temp: temp, t°, t:, tª, c°
🫁 Resp: fr, resp, rr, rpm
❤️ Pulso: fc, hr, bpm, latidos
🩸 Gluc: gluc, glucosa, hgt, azúcar
🩺 TA: ta, presión, bp, pa, t/a
☁️ Sat O2: sat, spo2, o2, oxígeno
⚖️ Peso: peso, kg, wt, weight
📏 Talla: talla, ht, height, estatura
📑 Resultados: resultados, estudios, auxiliares
💊 Tratamiento: tratamiento, plan, tx, indicaciones
🔮 Pronóstico: pronostico, px, evolucion

🚀 Inicio Rápido

  • Llenado: Ingresa datos o usa Importar desde notas.
  • Aplicar: Clic en ⚡ Aplicar en Azura o Ctrl + Enter.
  • Auto-Fill: Abre Azura y el script hará el resto.

📂 Gestión de Datos

  • Sesiones: Guarda y renombra sesiones en la barra lateral.
  • JSON: Usa 📁 Importar JSON para cargar archivos.
  • Limpieza: Limpiar texto ✨ normaliza tus notas.
